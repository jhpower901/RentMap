🔔 알림 설정

Discord 또는 Slack Webhook URL을 등록하면 원하는 조건의 매물이 변동될 때마다 알림을 받을 수 있습니다.
최대 5개까지 등록 가능합니다.

Discord: 채널 설정 → 연동 → 웹후크 → 새 웹후크에서 URL 복사
Slack: Incoming Webhooks 앱에서 채널 추가 후 URL 복사
면적 필터 사용을 체크하면 내 면적 필터 폴리곤 안의 매물만 알림을 보냅니다.
